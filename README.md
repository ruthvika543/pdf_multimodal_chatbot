# Multimodal PDF Chatbot

A RAG-based chatbot that answers questions from PDF documents including text, images, and tables.

## Stack
- PyMuPDF — PDF parsing
- LangChain — chunking and RAG chain
- OpenAI GPT-4o — multimodal understanding
- Pinecone — vector storage
- Streamlit — UI
